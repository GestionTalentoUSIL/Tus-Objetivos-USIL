# 🎯 Tus Objetivos USIL - Generador de Objetivos SMART con IA

## ✅ **CONFIGURACIÓN SIMPLE (2 MINUTOS)**

Tu aplicación usa **Groq** (IA 100% gratuita, ultra rápida).

### **Paso 1: Obtener API Key de Groq (GRATIS)**

1. Ve a: **https://console.groq.com**
2. Regístrate con Google o Email (no necesita tarjeta 💳)
3. Ve a **API Keys** → **Create API Key**
4. Copia la key (empieza con `gsk_...`)

### **Paso 2: Configurar en el código**

1. Abre `index.html` con un editor de texto
2. Busca la línea ~60: `const GROQ_API_KEY = '';`
3. Pega tu key:
   ```javascript
   const GROQ_API_KEY = 'gsk_TU_KEY_AQUI';
   ```
4. Guarda (Ctrl + S)

### **Paso 3: Usar**

1. Abre `index.html` en tu navegador
2. Completa el formulario de 3 pasos
3. Click en "Generar 5 Objetivos con IA"
4. ✅ ¡Listo! Tus objetivos SMART aparecerán
