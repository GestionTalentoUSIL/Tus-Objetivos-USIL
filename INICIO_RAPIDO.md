## 🧠 OPCIÓN 2: CLAUDE (Mejor calidad - $5 gratis)

### Paso 1: Obtén tu API Key (2 minutos)

1. Ve a: **https://console.anthropic.com/**
2. Click en **"Sign Up"**
3. Usa tu email
4. Verifica tu cuenta
5. Ve a: **https://console.anthropic.com/settings/keys**
6. Click **"Create Key"**
7. Copia la key (empieza con `sk-ant-...`)

### Paso 2: Configúrala en el código (1 minuto)

1. **Botón derecho** en `index.html` → **"Abrir con" → "Bloc de notas"** (o tu editor favorito)

2. Presiona **Ctrl + F** (buscar)

3. Busca: `claude: {`

4. Encontrarás algo como:
   ```javascript
   claude: {
     key: '', // Usuario debe agregar su key de Anthropic
   ```

5. Cambia a:
   ```javascript
   claude: {
     key: 'sk-ant-api03-TU_KEY_AQUI', // 👈 PEGA TU KEY AQUÍ
   ```

6. **Guarda** el archivo (Ctrl + S)

### Paso 3: Úsalo

1. Abre `index.html` en tu navegador
2. Completa el formulario
3. Selecciona **"Claude"** (botón morado 🧠)
4. Click **"Generar con Claude"**
5. ¡Disfruta respuestas más profesionales!

## 🛠️ SOLUCIÓN RÁPIDA DE PROBLEMAS

### ❌ Error: "NO_KEY"
**Solución:** Configura la API key del proveedor seleccionado

### ❌ Error: "models/gemini-1.5-flash not found"
**Solución:** Ya está corregido - actualiza el archivo index.html

### ❌ Error: "Invalid JSON"
**Solución:** Ya está corregido - el nuevo código parsea mejor

### ❌ No aparece nada
**Solución:** 
1. F12 → Console
2. Busca errores rojos
3. Comparte el error conmigo

### ❌ Genera solo 3 objetivos en lugar de 5
**Solución:** Esto pasa a veces con IA
- Click "← Atrás" 
- Click "Generar" de nuevo
- Prueba con otro proveedor

### ❌ Los objetivos no son buenos
**Solución:**
- Llena el formulario con MÁS detalle
- Sé específico en responsabilidades y KPIs
- Prueba con Claude (mejor calidad)
